# Calculator Project

A simple calculator program written in py++.

## Features

- Basic arithmetic operations (add, subtract, multiply, divide)
- Error handling for division by zero
- Clean output formatting

## Running

```
python run.py src/main.pypp
```

Or:

```
pypp run src/main.pypp
```

## Building

```
pypp build .
```

This will create a `build/` directory with compiled files.
